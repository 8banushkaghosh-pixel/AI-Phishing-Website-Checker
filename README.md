# AI-Powered Phishing Website Checker

## Overview
The AI-Powered Phishing Website Checker is a web application designed to analyze website URLs and identify potential phishing websites using common phishing detection techniques. The application checks URLs for suspicious characteristics and provides a risk assessment to help users avoid malicious websites.

## Features
- Detects URLs containing IP addresses
- Checks for HTTPS security
- Identifies suspicious symbols such as '@'
- Detects excessive hyphens in URLs
- Flags unusually long URLs
- Classifies websites as Safe, Suspicious, or High Risk
- Simple and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Enter a website URL.
4. Click the **Check** button.
5. View the phishing risk result.
